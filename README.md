Descripción:Prototipo funcional de una plataforma de CAA dirigida a niños no verbales, que permite la comunicación mediante pictogramas en una interfaz accesible.

Acceso a datos:
- Da clic en el ícono de engranaje (se encuentra a la mitad a la derecha)
- Ingresa la contraseña: 1234
Sobre el almacenamiento: Los datos se guardan localmente en el navegador (LocalStorage), registrando conteo de pictogramas y horario en el que se usó.

Características:
- Interfaz basada en pictogramas
- Organización en cuadrícula
- Construcción de mensajes
- Registro de uso (conteo y tiempo)

Tecnología/Lenguaje:
- HTML
- CSS

Funcionamiento: El usuario selecciona pictogramas para construir mensajes, mientras el sistema registra las interacciones realizadas.
Estado: Prototipo en desarrollo
